Introduction

From the ezBtc API page: 1

We are currently working towards a full API release. In the meantime, we offer a read-only API. Current supported trading pairs: "xbtcad"

An account with ezBtc is not required to access the API methods.

The methods availabe are:

  • Ticker
  • Order Book
  • Completed Trades
